My Projects

weather-app

This Weather Web App is designed to provide real-time weather updates with a sleek, Google-like UI. Users can search for any city to get current weather conditions, including temperature, wind speed, humidity, and weather description. The app dynamically fetches data from the OpenWeatherMap API and updates the UI with relevant weather icons.

Stopwatch Web

This Stopwatch Web Application is a feature-rich time-tracking tool built using HTML, CSS, and JavaScript. It allows users to start, pause, reset, and record lap times efficiently. With a clean and modern UI, this stopwatch is designed to provide precise millisecond accuracy while ensuring a smooth user experience.

Tic Tac Toe web

This Tic-Tac-Toe Web Application is an interactive and visually appealing game built with HTML, CSS, and JavaScript. It allows two players to compete, taking turns to place their marks (X or O) on a 3x3 grid. The game includes real-time updates, win detection, and a reset option, making it an engaging experience for players.
